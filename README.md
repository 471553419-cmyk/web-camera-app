 📸 H5 Web Camera App

这是一个基于 HTML5 + JavaScript 的纯前端网页相机应用。

✨ 功能特点
实时预览：调用手机后置摄像头。
倒计时拍摄：点击开始后，会有 3-2-1 倒计时动画。
模拟取景框：带有聚焦效果的遮罩层。
照片处理：支持重新拍摄。
保存功能：支持长按保存照片到手机相册。

点击下方链接在手机上体验（需授权摄像头）：

🛠️ 技术栈
HTML5
 CSS3 (Flexbox)
 JavaScript (ES6, getUserMedia API, Canvas API)

 ⚠️ 注意事项
由于浏览器的安全策略，本应用必须运行在 HTTPS*环境或 Localhost*下才能成功调用摄像头。
